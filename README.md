# VORTEXv1.1 // Authentication Credential Breaker


VORTEX v1.1 is a simple tool built to help you test how strong a login system is against password guessing. It's designed to be super easy to use, so you can focus on learning cybersecurity in a safe and legal way.

Why use VORTEX?

  Fast: It uses multi-threading, so testing thousands of passwords feels like a breeze.

   Easy to read: It shows you the results (like 200 OK or 401 Unauthorized) in real-time, so you don’t have to dig through messy logs.

   Lightweight: Built with Python, it’s super smooth to run on Kali Linux or any standard machine.

   cd VortexAuth
   ```

```
